# SF Symbols Icon Mapping

## Standard Actions

| Action | Symbol |
|---|---|
| Menu | `line.3.horizontal` |
| Profile | `person.circle.fill` |
| Back | `chevron.left` |
| Forward / Disclosure | `chevron.right` |
| Search | `magnifyingglass` |
| Write / Compose | `square.and.pencil` |
| Close | `xmark` |
| Settings | `gearshape` |
