## String Rules

- Button labels: concise, 2-4 characters (e.g. `"등록"`, `"취소"`, `"로그인"`)
- Section headers: short but descriptive (e.g. `"MY 라임방"`, `"인기 게시글"`)
- Placeholder text: polite form with 합니다/세요 ending
